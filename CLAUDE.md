# ChromaSense — CLAUDE.md

## What this is

A single-file color discernment game hosted on GitHub Pages.
Live: https://spicysouvlaki.github.io/pantone/
Repo: https://github.com/spicysouvlaki/pantone

## Constraints (non-negotiable)

- **Single file** — everything lives in `index.html`. No build step, no npm, no CDN.
- **Zero dependencies** — vanilla HTML + CSS + JS only.
- **< 50 KB** — current size is ~48 KB. Check with `wc -c index.html` before pushing.
- **No backend** — `localStorage` only for persistence.

## Architecture

All code is in one `<script type="module">` block inside `index.html`.

### Color science
- Internal color space: **CIELAB (L\*a\*b\*)** for all math
- Rendering: **`oklch()`** CSS (converted from Lab via XYZ → OKLab), with `rgb()` fallback detected via `CSS.supports`
- Perceptual distance: **CIEDE2000 ΔE** (full Sharma et al. 2005 formula, not ΔE76)
- Gamut clamping: binary search on chroma until sRGB in-gamut, hue preserved

### Hue regions (8 total — FM100 structure)
| # | Name | Lab hue range |
|---|------|--------------|
| 0 | Red | 340°–20° (wrap) |
| 1 | Orange | 20°–50° |
| 2 | Yellow | 50°–80° |
| 3 | Yellow-Green | 80°–110° |
| 4 | Green | 110°–150° |
| 5 | Cyan | 150°–210° |
| 6 | Blue | 210°–270° |
| 7 | Purple | 270°–340° |

**Hard regions** (higher floor ΔE = 2.5): Yellow (2), Yellow-Green (3), Cyan (5).
All others: floor ΔE = 1.5.

> **Do not add a neutral/grey region here.** Greys are not part of the FM100 test —
> that test is purely chromatic hue discrimination at fixed lightness and chroma.
> A lightness discrimination mode is planned separately (see below).

### Difficulty
- Starts at ΔE = 20, shrinks × 0.85 on correct, × 1.4 on wrong
- Floor enforced per region (1.5 or 2.5)
- "Odd" swatch generated by binary-searching a hue rotation until CIEDE2000 ≈ target ΔE

### Game state
- 3 lives, 4.5s timer per round
- Score: `100 × timeFactor × (20/ΔE) × streakMultiplier`
- Regions rotate in shuffled batches of 8

### Screens
`idle → playing → (game over)`

### Player ranks (by best ΔE achieved)
| Rank | Best ΔE |
|------|---------|
| SAVANT | ≤ 2.2 |
| GIFTED | ≤ 4.5 |
| AVERAGE | ≤ 8.0 |
| FAILING ARTIST | ≤ 13.5 |
| POOR | > 13.5 |

### Storage key
`chromasense_v2` in localStorage.
